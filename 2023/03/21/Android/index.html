一.Button

1.点击事件设置

1.1 通过对Button对象创建点击事件的监听接口

// Button对象的点击事件监听，new一个View类中的一个名为OnClickListener接口的实现类，实现该接口的onClick方法
mybutton.setOnClickListener(new View.OnClickListener() {
    @Override
    public void onClick(View v) {
        myBtn_two.setText("按钮2已被点击");
    }
});

1.2 创建自定义方法，在Button标签中指定

public void click(View v) {
    myBtn_one.setText("按钮1已被点击");
}    
<Button
        android:id="@+id/btn_one"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:onClick="click"
        android:text="按钮1" />

1.3 对外类创建统一的监听接口

外类实现View.OnClickListener接口，实现onClick抽象方法

public class Button2Activity extends AppCompatActivity  implements View.OnClickListener{
    private Button myBtn_one;
    private Button myBtn_two;
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.button);
        myBtn_one = (Button) findViewById(R.id.btn_one);
        myBtn_two = (Button) findViewById(R.id.btn_two);

        myBtn_one.setOnClickListener(this);
        myBtn_two.setOnClickListener(this);
    }
    //实现onClick()接口
    @Override
    public void onClick(View v) {
        switch (v.getId()){
            case R.id.btn_one:
                myBtn_one.setText("1已被点击");
                break;
            case R.id.btn_two:
                myBtn_two.setText("2已被点击");
                break;
        }
    }
}
